Запусакемый код находится в файле "ConsoleApplication2.cpp",
который находится в папке "ConsoleApplication2/ConsoleApplication2" 

Чтобы запустить игру, необходимо скачать и распаковать саму папку "ConsoleApplication2".
Затем необходимо открыть Visual Studio, нажать на файл -> открыть решение -> выыбрать папку
"ConsoleApplication2/ConsoleApplication2" -> выбрать файл ConsoleApplication2.cpp.

После этого у вас на экране появится запускаемый код.
